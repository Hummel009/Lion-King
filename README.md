Обновлённый, слегка переписанный, оптимизированый код мода "Lion King" для Minecraft 1.6.4. Перенесён с Forge на Voldeloom и пригоден к разработке на момент 2023 года.

Для запуска и редактирования кода потребуется среда разработки Java и JDK. Подойдёт IntelliJ IDEA или Eclipse IDE, а также JDK 11.
